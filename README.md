# TianyiTap
一个支持多音源的 Mikutap 修改版。是 HFIProgramming/mikutap 之分支。
   
![](https://i.loli.net/2017/12/23/5a3e0812758da.gif)

# 修改说明  
- 增加了洛天依声源
- 增加了Warma声源
- 增加了星尘（Infinity）的声源
- 增加了不同声源选择功能（你也可以交pr增加新的声源！）
- 增加了镜音连、镜音铃（Kagamine Len & Rin）的声源
